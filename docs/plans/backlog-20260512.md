## 2. Global Definition of Done

每個 Epic 完成前必須符合：

1. 對應 PRD / test-spec 已存在於 `docs/plans/`。
2. 若涉及架構、依賴、schema、輸入政策，必須有 `docs/adr/YYYYMMDD-*.md`。
3. 至少一個 red test 或 fixture contract 先失敗，再進入實作。
4. 完整驗證通過：

```bash
uv run pytest -q
uv run guitar-tab-generation --help
```

5. 若涉及輸出 artifact，必須跑對應 CLI fixture 到 `/tmp`。
6. 不得追蹤 `.omx/`、`__pycache__/`、`.pytest_cache/`、大型輸出、下載媒體。
7. Commit 不加 `Co-authored-by: OmX ...`。
