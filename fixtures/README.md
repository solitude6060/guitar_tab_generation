# Golden MVP fixtures

These fixtures are synthetic, self-made WAV clips for legal local-audio-first MVP testing. Each clip is 30 seconds long and has paired metadata plus a manual rubric record.

- `simple_chords_30_90s.wav`: synthetic chord-strum proxy.
- `single_note_riff_30_90s.wav`: synthetic single-note riff proxy.
- `single_note_lead_30_90s.wav`: synthetic single-note lead proxy.

The metadata files are the current baseline truth for deterministic MVP tests; future real recordings can replace these only if rights and rubric records remain present.
